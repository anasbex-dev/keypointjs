# Code of Conduct — KeypointJS

## Our Pledge

In the interest of fostering an open, professional, and respectful environment,  
we as contributors, maintainers, and users of **KeypointJS** pledge to make participation
in this project a harassment-free experience for everyone.

KeypointJS is a technical project focused on **security, architecture, and API design**.
All discussions should remain **constructive, technical, and respectful**.

---

## Our Standards

Examples of behavior that contribute to a positive environment include:

- Using respectful and professional language
- Giving constructive feedback focused on code, architecture, or design
- Respecting different technical opinions and approaches
- Accepting criticism gracefully and discussing it technically
- Helping others understand the framework without condescension

Examples of unacceptable behavior include:

- Personal attacks, insults, or derogatory comments
- Harassment, threats, or hate speech of any kind
- Dismissing ideas without technical reasoning
- Trolling, flame wars, or intentionally derailing discussions
- Pressuring maintainers to accept changes without discussion

---

## Scope

This Code of Conduct applies to all project spaces, including but not limited to:

- GitHub issues
- Pull requests
- Discussions
- Commit messages
- Community conversations related to KeypointJS

---

## Enforcement

Project maintainers are responsible for clarifying standards of acceptable behavior
and will take appropriate and fair corrective action in response to any instances of
unacceptable behavior.

Actions may include:

- Warning the offending party
- Temporarily limiting participation
- Closing issues or pull requests
- Permanent ban from the project spaces (for severe or repeated violations)

---

## Maintainer Authority

KeypointJS is currently maintained by its creator.

Final decisions regarding architecture, security design, and project direction
remain with the maintainer to ensure consistency, stability, and security.

Disagreement is welcome — disrespect is not.

---

## Reporting

If you experience or witness behavior that violates this Code of Conduct,
please report it privately to the maintainer.

Contact:
 **anasbex-dev (GitHub)**
